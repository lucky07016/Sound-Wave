# 🎵 Sound Wave

Sound Wave is a modern and responsive music web application built using **HTML, CSS, and JavaScript**.  
It provides an interactive UI to play, control, and explore audio in a smooth and engaging way.

---

## 🚀 Features

- 🎧 Play / Pause music  
- ⏭️ Next / Previous track controls  
- 🔊 Volume control  
- 📊 Progress bar for tracking audio  
- 🎨 Responsive and modern UI  
- ⚡ Smooth user experience  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
